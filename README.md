Project 1 (Selection, Manipulation, and Travel) requirements : 
0. storyboard
1. initalizing: Place the user in the center of the tent (5 points)
2. Spawning: Allow the users to spawn two different items of your choices (3 + 2 = 5
points), and provide a way for the user to modify the placement and orientation (5
points for placement and 5 points for orientation). Once spawned, the objects should
have physics properties attached to them and react to gravity and collisions (5
points)
3. Selection and Manipulation: Allow the users to select already placed items in the
space and highlight the corresponding selections (5 points for selection, 5 points for
highlight). We expect two different interaction schemes for selection (5 points for the
second selection method). Allow manipulation of the highlighted objects. After an
item is selected, design a way to allow the users to move, rotate and scale the item
(3 + 3 + 4 = 10 points). Only the trigger button and the grip button are allowed to be
used. (-5 out of 10 if other buttons are used for manipulation)
4. Travel: The user should be able to move around the space allowing free movement
from one side to the other and control over orientation (5 points movement, 5 points
orientation)
5. Indicator : It is not necessary to use a ray, but you should include an
indicator/pointer for each corresponding selection and travel method (5 points for
selections, 5 points for travel).
6. Heuristic Evaluation (20 points): Once you have finished implementing your project, you should conduct a heuristic evaluation of the system using the template in your Project Design Document. There are a total of 10 heuristics (worth 2 points each) for which you need to input:
a. Violation + Rationale: Does your system violate the heuristic? Why do you think
so? (1 point each)
b. Severity: How much does this violation impact the user experience? (0.25 points
each)
c.Recommendation: How can you fix this issue? If any of your techniques from the
design exploration work better, mention it. (0.75 points each)
